# Example
任務：
- 實作 CSV 資料列計數功能。

功能要求：
- 使用方式為 `python -m src.analyze <csv_path>`。
- 成功時輸出 `rows: <資料列數>`，不包含標題列。
- 檔案不存在或無法讀取時，將簡潔錯誤輸出至 stderr。
- 失敗時使用非零 Exit code，且不顯示 traceback。

限制：
- 使用 Python 標準函式庫。
- 不修改 `data/sample.csv`。
- 不做無關重構。
- 遵守 `AGENTS.md`。
- 本次不 commit、不 push。

驗證：
- 新增成功案例與檔案不存在案例的測試。
- 執行相關測試。
- 實際執行成功與失敗的 CLI 案例。

完成後只回報修改檔案、驗證結果與尚存問題。
